再次更新文章

1.新建一篇文章，在网站根目录：D:\Hugo\Sites\www.mdrink.cn

hugo new post/第二篇测试博客.md

2.在目录 D:\Hugo\Sites\www.mdrink.cn\content\post 下找到对应文件进行修改

3.本地预览, http://localhost:1313/

hugo server –theme=maupassant –buildDrafts

4.构建 Hugo 网站

hugo –theme=maupassant –baseUrl=“http://www.mdrink.cn/"

5.切换到目录 D:\Hugo\Sites\www.mdrink.cn\public ，提交修改至本地库

更新推送 可能会失败，先比较这两个 git pull origin master git config –global http.sslverify “false”

git add -A git commit -m ‘commit info’

6.将修改推至远程库

git push -u origin master

第一次新建是以下 cd public git init git remote add origin https://github.com/meijinguang/meijinguang.github.io.git git add -A git commit -m “first commit” git push -u origin master
